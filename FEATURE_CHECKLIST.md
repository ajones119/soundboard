# Soundboard Feature Checklist

## Critical for Sound Layering

- [x] **Master Volume Control**
  - Global slider to adjust all playing sounds proportionally
  - Useful for quick overall adjustments

- [ ] **Fade In/Out on Play/Stop**
  - Smooth fade when starting/stopping sounds
  - Prevents jarring audio cuts
  - Optional fade duration control

- [ ] **Stop vs Pause Distinction**
  - Stop: Reset to beginning
  - Pause: Resume from current position
  - Useful for one-shots vs looping ambiance

## Organization & Quick Access

- [x] **Categories/Tags**
  - Tag sounds (e.g., "forest", "tavern", "battle", "weather", "magic")
  - Filter library by category
  - Group saved sounds by category

- [ ] **Presets/Sound Sets**
  - Save combinations of sounds with volumes/loops as presets
  - Quick load: "Tavern Ambiance", "Forest at Night", "Battle Scene"
  - One-click to load a preset

- [ ] **Quick Mute All**
  - Emergency silence button (separate from pause)
  - Instantly mute everything

## Visual Feedback

- [ ] **Visual Playing Indicators**
  - Visual indicator (e.g., pulsing border, waveform) on playing sounds
  - At-a-glance view of what's active

- [ ] **Volume Level Meters**
  - Small VU meters per sound
  - Shows actual audio output level

## Advanced Playback

- [ ] **Crossfade Between Sounds**
  - Fade out one sound while fading in another
  - Smooth transitions

- [ ] **Playback Speed/Pitch**
  - Slow down/speed up sounds
  - Pitch adjustment (useful for variations)

## DM-Specific Features

- [ ] **Sound Preview on Hover**
  - Quick preview without adding to saved sounds
  - Helps find the right sound quickly

- [ ] **Keyboard Shortcuts**
  - Spacebar: pause all
  - Number keys: play sound at that position
  - Quick access without mouse

- [ ] **Sound Normalization**
  - Auto-adjust volumes so sounds are roughly the same level
  - Prevents one sound from overpowering others
